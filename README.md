# AC to DC Converter PCB Design

## Overview

This repository contains the complete PCB design of an AC to DC Converter developed using **KiCad 9.0**. The project includes the circuit schematic, PCB layout, and fabrication-ready Gerber files required for PCB manufacturing.

The PCB design demonstrates the complete hardware development workflow, including schematic capture, PCB layout design, component placement, routing, design rule verification, and Gerber file generation.

---

## Project Objectives

- Design an AC to DC Converter PCB.
- Develop the circuit schematic using KiCad.
- Create a compact and efficient PCB layout.
- Generate fabrication-ready Gerber and drill files.
- Organize the project files for manufacturing and future modifications.
